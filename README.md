# 🌿 Туған елке табиғаты — Қызылорда экожүйелері

Қызылорда облысының бірегей табиғи экожүйелерін көрсететін виртуалды экскурсия сайты.

## Технологиялар

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Leaflet / React-Leaflet** (интерактивті карта)
- **Framer Motion** (анимациялар)

## Бөлімдер

- 🌊 **Арал теңізі** — экологиялық апат пен қалпына келу тарихы
- 🌿 **Сырдария өзені** — тоғай экожүйесі, флора мен фауна
- 🏜️ **Қызылқұм шөлі** — шөл жануарлары мен өсімдіктері
- 🗺️ **Интерактивті карта** — барлық аймақтар картада
- 📊 **Ғылыми деректер** — статистика және хронология

## Іске қосу

```bash
# Тәуелділіктерді орнату
npm install

# Даму режимі
npm run dev

# Өндіріс build
npm run build
npm start
```

Браузерде `http://localhost:3000` адресін ашыңыз.

## Файл құрылымы

```
kyzylorda-eco/
├── app/
│   ├── data/
│   │   └── ecosystems.ts     # Барлық деректер
│   ├── globals.css           # Глобалды стильдер
│   ├── layout.tsx            # Негізгі layout
│   └── page.tsx              # Басты бет
├── components/
│   ├── Navbar.tsx            # Навигация
│   ├── Hero.tsx              # Басты баннер (canvas анимация)
│   ├── EcosystemsSection.tsx # Экожүйе бөлімі
│   ├── EcosystemCard.tsx     # Экожүйе карточкасы
│   ├── InteractiveMap.tsx    # Leaflet карта
│   ├── Gallery.tsx           # Галерея
│   ├── FactsSection.tsx      # Деректер + хронология
│   └── Footer.tsx            # Футер
├── tailwind.config.ts
├── tsconfig.json
└── next.config.js
```

## Өзгертулер

Барлық контент `app/data/ecosystems.ts` файлында. Жаңа экожүйе немесе карта нүктесін осы файлда қосуға болады.
